# STEP 1 – Project Foundation & Global Layout

You are a senior Front-End Developer, UI/UX Designer, SEO Specialist, and Performance Engineer.

Build **Step 1** of a premium seafood business website.

Do **not** build the Hero Section, Product Categories, Today's Catch, Gallery, or any other content sections yet.

The goal of Step 1 is to create the **reusable foundation** that every page of the website will share.

---

# Technology Stack

Use only:

* HTML5
* CSS3
* Vanilla JavaScript (ES6)
* CSS Grid
* Flexbox
* Swiper.js (Only link the library, do not build the slider yet.)
* AOS (Only link the library, do not add animations yet.)
* Font Awesome
* Google Fonts (Poppins)

Do NOT use:

* Bootstrap
* React
* Vue
* Angular
* Tailwind CSS
* jQuery
* Any backend technology

The website must work perfectly on GitHub Pages and Vercel.
